## Introduction
*2d Cutout Masks* is a shader for unity, that uses masks to cutout sprites. 

### Features
 - Easy to use
 - Any shape
 - Alpha
 - Sprite color
 - Unity Animations
 - Rotation
 - Transparency
 - Pixel snap
 - Unlimited number of masks
